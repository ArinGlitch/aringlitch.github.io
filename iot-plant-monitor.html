IoT Based Plant Monitoring System

This project is a smart IoT system designed to monitor soil moisture, temperature, humidity, and light exposure for plants. It helps users keep their plants healthy by providing real-time data and alerts.

Features:

  • Soil moisture and temperature sensors
  • Real-time dashboard for readings
  • Mobile notifications when plants need watering
  • Arduino + NodeMCU + Firebase integration

Tech Stack:

Arduino, ESP8266, Firebase Realtime Database, HTML, CSS, JavaScript
